# Random Course Generator (ICS 385 Week 6 / Assignment 6b)
## Overview
Simple string randomizer using Express.js and EJS

## Setup
- cd (cwd)
- npm install
- npm update

## How to Run
node index.js

## New Features Added
1) made course ID and course name lists (using ref at .txt eof)
2) moved lists to their own file
3) custom middleware to read files
3) additional logic to label some classes as "Intro" or "Capstone"
